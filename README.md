# Lucy Chapar

Portfolio and creator home base for `lucychapar.com`.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The cPanel deployment file copies `dist/` into `/home/lucymrsy/public_html/`.
For Namecheap/cPanel Git deployment, build locally and commit the generated
`dist/` folder before deploying from cPanel.
